# Final-Project-Guild-Management-API

## Description
This project is a REST API for managing game guilds, members, and groups.

## Resources
- Guilds
- Users
- Groups

## Endpoints

### Guilds
GET /guilds  
POST /guilds  
PUT /guilds/:id  
DELETE /guilds/:id  

### Users
GET /users  
POST /users  

### Groups
GET /groups  
POST /groups  

## Run Project

install dependencies

npm install

run server

node server.js
